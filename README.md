# Pup-Collar
Arduino and NRF24L01 based radio dog collar.

To help easily locate dog when walking off leash.
Collar blinks periodically and when remote button is pressed collar beeps. 

Project components:
  Arduino Nano board x2
  NRF24L01+ 2.4GHz transceiver module x2
  Piezo buzzer
  RGB LED
  Pushbutton
  220 Ohm resistor x2
  9V battery x2
